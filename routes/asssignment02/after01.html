Responsive Web Design

 This design philosophy can be integrated within either of the earlier concepts. Responsive Web Design (RWD) is simply dynamic rendering based on device size. Websites and applications that incorporate RWD principles have prepared for the end-user to be using anything from a desktop, to tablet, or smartphone. The most common implementation of RWD is through CSS media queries. @media is a rule within CSS that changes or applies a look or layout when specified parameters are met such as width, height, orientation, scripting, etc.

 In 2015, for the first time in history, users with mobile devices (smartphones and tablets) out-searched the web compared to users with desktops and laptops. Google embraced this turn with an update to it's search algorithm, coined " Mobilegeddon ", that slightly favored websites which could return appealing layouts to mobile users. Searches from laptops and desktops were not affected. The resulting affect to traffic on non-responsive sites was fairly insubstantial, but could spell a trend in other markets as mobile performance increases and the desktop environment user-base shrinks.
